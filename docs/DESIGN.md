# Дизайн-система SIGMA ZADROTS

Стилистика взята с вывески клуба «SIGMA CLUB»: киберпанк-неон.

## Палитра

| Токен | HEX | Назначение |
|---|---|---|
| `bg` | `#070310` | основной фон (почти чёрный с фиолетовым) |
| `bg-soft` | `#0d0718` | фон карточек/панелей |
| `bg-elev` | `#140b22` | приподнятые поверхности, hover |
| `neon-magenta` | `#ff2bd6` | главный неон (рамки, акценты, лого) |
| `neon-purple` | `#b026ff` | вторичный неон, свечение |
| `neon-cyan` | `#2de2ff` | акцент для данных/ссылок/доп. свечения |
| `text` | `#f4ecff` | основной текст |
| `text-dim` | `#b9a9d6` | приглушённый текст |

## Шрифты (Google Fonts)

- **Orbitron** (700/900) — латинские заголовки и лого `SIGMA ZADROTS`.
- **Exo 2** (400/600/700) — основной текст и кириллица (UI на русском).
- Заголовки: UPPERCASE, широкий letter-spacing.

## Эффекты

- **Неоновая рамка**: 1–2px рамка цвета неона + многослойный `box-shadow`
  (внутреннее + 2–3 внешних слоя разного радиуса) для свечения.
- **Неоновый текст**: `text-shadow` несколькими слоями того же оттенка.
- Карточки: `border-radius: 16px`, тёмная заливка, неоновая обводка.
- Заголовки секций — «пилюли»/чипы с неоновой рамкой (как PC STANDARD / VIP ROOM).
- Лёгкий flicker/pulse на лого (опционально, аккуратно, уважать `prefers-reduced-motion`).
- Фон: тёмный градиент + едва заметная сетка/скан-линии.

## Компоненты

- `NeonCard` — карточка с неоновой рамкой и свечением (вариант цвета через проп).
- `NeonChip` — заголовок-пилюля секции.
- `NeonLogo` — лого SIGMA ZADROTS (Orbitron, двойное свечение).
- Лидерборд — тёмная таблица, неоновые разделители, подсветка строки на hover,
  топ-3 с усиленным свечением.
